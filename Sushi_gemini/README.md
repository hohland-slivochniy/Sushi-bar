# 🌸 Moon Song (Мун Сонг) — Доставка японской кухни премиум-класса

Визуальный стиль проекта:
- **Пастельная палитра цветов (Pastel Palette):** нежно-розовый (`#ffd6e0`), молочно-персиковый (`#ffe5ec`), лавандовый (`#e8dafb`), мятный (`#d8f3dc`).
- **Liquid Glass Style (Стеклянный реализм):** полупрозрачные карточки с размытием фона (`backdrop-blur-xl`), мягкие блики и внутренние тени.
- **Скругленные углы:** закругления `rounded-3xl` и плавные линии интерфейса.

---

## 🛠 Технологический стек
Проект реализован по модульной архитектуре с поддержкой двух режимов работы:
1. **Frontend SPA:** HTML5, CSS3, JavaScript / TypeScript (React + Vite + Tailwind CSS + JSS).
2. **Backend API:** Python 3.10+, FastAPI, SQLAlchemy, SQLite 3 (реляционная база данных в файле `.db`).

---

## 📋 Пользовательские сценарии в приложении

### 1. Просмотр меню и добавление в корзину (Главная страница)
- В правом верхнем углу находится кнопка перехода на страницу регистрации и логотип **Moon Song**.
- Меню разделено на **9 категорий**:
  1. **Сеты** (Сет «Все включено», Авторский сет, Сет Moon, Сет Sun, Мини сет, Куриный сет, Сет креветки, Сет горячих роллов, Сет сладкий)
  2. **Суши и Гунканы** (Суши с креветкой, Суши с лососем, Суши горячие)
  3. **Роллы** (Роллы по-домашнему, Авторские роллы, Куриные роллы, Роллы в блине, Запеченный ролл с лососем, Запеченный ролл с курицей, Запеченный ролл с грибами, Роллы с огурцом, Роллы с креветкой, Роллы с красной рыбой)
  4. **Воки** (Вок с курицей, Вок с морепродуктами, Вок острый, Вок с грибами)
  5. **Донбури** (Донбури с креветками, Донбури с курицей, Донбури с терияки)
  6. **Супы** (Том ям с курицей, Том ям с креветками, Том ям с мисо, Суп рамен)
  7. **Напитки** (Лимонад «Буратино», Бабл ти, Егермейстер безалкогольный, Чай, Кофе, Энергетик)
  8. **Десерты** (Чизкейк, Маффин шоколадный, Батончик с вишней, Булочка с маком, Мороженое, Шоколадный фондан)
  9. **Прочее** (Салфетки, Палочки, Имбирь, Соус, Васаби, Рюмочка, Брелок с луной)
- **Карточка товара в реализме:** Отображает цену, вес, ккал, значки халяль/остроты/сладости, список ингредиентов через запятую и высококачественное фото.
- **Плавное увеличение:** При клике на карточку она плавно увеличивается за 0.5 секунд, фон сайта размывается, открывается большое окно с подробным описанием и рекомендованными напитками.
- **Счетчик в корзине:** При добавлении товара в правом верхнем углу карточки появляется нежно-розовый кружок с белым числовым счетчиком.

### 2. Регистрация и Личный Кабинет (.db хранилище)
- Окно слева с вводом данных (**Имя, Фамилия, Отчество, Почта**) и подсказками placeholder.
- При клике на «Зарегистрироваться» происходит автоматический вход в аккаунт с сохранением профиля в реляционной базе данных `.db`.
- В Личном кабинете отображается история заказов с деталями доставки и суммой чека, а также встроенный инспектор таблиц SQLite.

### 3. Просмотр корзины и оформление заказа
- При нажатии на иконку корзины открывается всплывающее окно по центру экрана. В нем доступен список позиций, изменение количества кнопками `+` и `-` и подсчет итоговой суммы.
- По нажатию на «Заказать» открывается форма ввода адреса (Улица, Дом, Подъезд, Квартира, Комментарий).
- После подтверждения отображается праздничное окно «Вы успешно оформили заказ!» с конфетти, а корзина очищается.

## 🚀 Как запустить проект (Python / FastAPI Backend)

### 1. Установка зависимостей
Откройте терминал в папке проекта и установите библиотеки из файла `requirements.txt`:

```bash
pip install -r requirements.txt
```

### 2. Запуск сервера FastAPI
Выполните команду старта сервера с автоперезагрузкой:

```bash
uvicorn backend.main:app --host 0.0.0.0 --port 8000 --reload
```

После запуска сервер создаст или подключит файл базы данных `moonsong.db` и будет доступен по адресу:
- 👉 **http://localhost:8000** — Главная страница
- 👉 **http://localhost:8000/docs** — Интерактивная документация REST API (Swagger UI)

> **Примечание:** Также прямо в интерфейсе веб-сайта доступна кнопка **«Скачать Python Backend (.ZIP)»**, генерирующая полный архив с кодом проекта на Python и FastAPI!
